# Package Management

## APT (Debian/Ubuntu/Termux)
Установка: <run>apt install package</run>
Поиск: <run>apt search keyword</run>
Обновление списка: <run>apt update</run>
Обновить всё: <run>apt upgrade -y</run>
Удалить: <run>apt remove package</run>
Информация: <run>apt show package</run>

## PIP
Установка: <run>pip install package</run>
Поиск: <run>pip search keyword</run>
Информация: <run>pip show package</run>

## NPM
Установка глобально: <run>npm install -g package</run>
Установка локально: <run>npm install package</run>
Запуск скрипта: <run>npm run script</run>

## Brew (macOS)
Установка: <run>brew install package</run>
Поиск: <run>brew search keyword</run>
Информация: <run>brew info package</run>

## Cargo (Rust)
Установка: <run>cargo install package</run>
Поиск: <run>cargo search keyword</run>
